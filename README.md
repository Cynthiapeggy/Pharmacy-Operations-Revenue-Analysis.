# Pharmacy-Operations-Revenue-Analysis.
## Project Overview
This project analyzes pharmacy sales data using Microsoft Excel to uncover revenue trends, top-performing drugs, and customer purchasing behavior. The dashboard provides interactive insights to support inventory management, supplier performance tracking, and sales optimization decisions.

## Dataset used
<a href="https://github.com/Cynthiapeggy/Pharmacy-Operations-Revenue-Analysis./blob/main/PHARMACY.xlsx">Dataset</a>
## Key Performance Indicators (KPIs)
There are six KPI cards to highlight the most important pharmacy metrics:
- Total Revenue
- Total Transactions
- Avg Sales per
- Total Quantity Sold
- Revenue per Unit
- Average Unit Price
